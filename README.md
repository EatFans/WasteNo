## 背景
大学里总有那么一些课程：
- 教学内容多年未变，照本宣科，与真实行业需求脱节；
- 课堂互动流于形式，“水”成为师生心照不宣的共识。

讽刺的是，这些内容质量有限的课程，却在技术限制上层层加码：
- 防切屏：离开页面超过几次就计零分；
- 禁复制：连合法的笔记整理都被一刀切；
- 禁粘贴：不允许从自己的文档里粘贴内容；
- 限时答题：时间短到只够抄答案，不够思考。

技术本应为教学服务，最终却变成了“管理学生”的工具。
## 立场

我们不评价教学水平，只解决一个实际问题：

- 我的时间，不应该被低质量的技术手段困住。
- WasteNo 不是为了作弊，是为了让工具回归工具，让学习回归学习。

## 功能

- 反制网站的禁止复制
- 反制网站的禁止粘贴
- 反制切屏检查
- 更多功能持续更新

> 注：后续版本继续更新更多反制网站限制的功能...

## 安装方式

### 开发者模式安装（未上架商店时）

1. 下载本仓库代码到本地（或 `git clone`）
2. 打开浏览器扩展管理页面：
    - Chrome：`chrome://extensions`
    - Edge：`edge://extensions`
    - 其他Chromium内核浏览器类似
3. 开启右上角 **“开发者模式”**
4. 点击 **“加载已解压的扩展程序”**
5. 选择本插件所在的文件夹（包含 `manifest.json` 的那一层）
6. 安装成功，工具栏出现 WasteNo 图标

## 使用方法

1. 点击浏览器右上角 WasteNo 图标
2. 根据界面开关，按需开启对应功能（如“解除复制限制”“绕过切屏检测”）
3. 大部分功能开启后**立即生效**，无需刷新页面
